## 📖 Deskripsi Program

Program ini adalah sistem manajemen top up E-Wallet berbasis Java.

Fitur:

* Tambah data top up
* Lihat data
* Hapus data

Jenis E-Wallet:

* DANA
* GOPAY

---

## 🧠 Konsep OOP

### ✅ Inheritance

Dana dan Gopay merupakan turunan dari EWallet.

---

### ✅ Polymorphism

#### 🔹 Overriding

Method `tampilkanInfo()` dioverride.

#### 🔹 Overloading

Method `topUp()` memiliki 2 bentuk:

* topUp(int jumlah)
* topUp(int jumlah, String bonus)
